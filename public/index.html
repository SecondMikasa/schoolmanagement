API Endpoints

POST https://schoolmanagement-fnrl.onrender.com/api/schools/addSchool

Add a New School

Adds a new school to the database with location information.

Request Body:

  • name String - Name of the school
  • address String - Physical address of the school
  • latitude Float - Latitude coordinate (-90 to 90)
  • longitude Float - Longitude coordinate (-180 to 180)

Response:

{
  "success": true,
  "message": "School added successfully",
  "data": {
    "id": 1,
    "name": "Example School",
    "address": "123 Education St",
    "latitude": 40.7128,
    "longitude": -74.006,
    "createdAt": "2025-05-13T10:30:00.000Z",
    "updatedAt": "2025-05-13T10:30:00.000Z"
  }
}
GET https://schoolmanagement-fnrl.onrender.com/api/schools/listSchools

List All Schools

Retrieves all schools sorted by proximity to the provided location.

Query Parameters:

  • latitude Float - User's latitude coordinate
  • longitude Float - User's longitude coordinate

Response:

{
  "success": true,
  "message": "Schools retrieved successfully",
  "data": [
    {
      "id": 1,
      "name": "Example School",
      "address": "123 Education St",
      "latitude": 40.7128,
      "longitude": -74.006,
      "createdAt": "2025-05-13T10:30:00.000Z",
      "updatedAt": "2025-05-13T10:30:00.000Z",
      "distance": 2.5
    },
    // More schools...
  ]
}
